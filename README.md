# Maple 记事本

一款基于 Vue 3 和 Canvas 技术开发的现代化思维导图编辑器，提供直观的可视化编辑体验。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 功能说明

### 节点操作

| 操作 | 快捷键/方式 | 说明 |
|------|------------|------|
| 创建根节点 | 点击画布空白处 | 在点击位置创建新的根节点 |
| 添加子节点 | `Enter` | 在选中节点下添加子节点 |
| 添加同级节点 | `Tab` | 在选中节点的父节点下添加新子节点 |
| 删除节点 | `Delete` / `Backspace` | 删除选中节点及其所有子孙节点 |
| 编辑节点文本 | 双击节点 | 弹出输入框编辑节点内容 |
| 左键拖拽 | 鼠标左键拖拽 | 移动节点及其所有子孙节点 |
| 右键拖拽 | 鼠标右键拖拽 | 只移动当前节点，不移动子节点 |
| 微调位置 | `↑` `↓` `←` `→` | 每次移动 10 像素 |

### 画布操作

| 操作 | 快捷键/方式 | 说明 |
|------|------------|------|
| 平移画布 | 按住 `空格键` + 拖拽 | 自由移动画布视图 |
| 放大画布 | 鼠标滚轮上滚 / `+` 键 | 以鼠标位置为中心放大 |
| 缩小画布 | 鼠标滚轮下滚 / `-` 键 | 以鼠标位置为中心缩小 |
| 适应屏幕 | 双击画布空白处 | 自动调整缩放和位置以显示所有节点 |

### 布局功能

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 切换布局方向 | `L` | 在「左右布局」和「上下布局」之间切换 |

**布局特点**：
- 智能树形布局算法，避免节点重叠
- 只对选中的节点及其子树应用新布局
- 平滑的节点移动动画
- 支持左右和上下两种布局方向

### 节点折叠/展开

- **折叠/展开节点**：双击节点右侧的圆形指示器（绿色表示已折叠，灰色表示已展开）
- 折叠指示器显示 `+` 或 `−` 图标

### 撤销/重做

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 撤销 | `Ctrl + Z` | 恢复上一步操作 |
| 重做 | `Ctrl + Y` 或 `Ctrl + Shift + Z` | 恢复被撤销的操作 |

### 连接线操作

- **选择连接线**：点击节点之间的连线（选中后显示绿色）
- **删除连接线**：选中连接线后按 `Delete` / `Backspace`

### 视觉效果

#### 节点样式
- **三色渐变背景**：提供层次感和现代感
- **柔和阴影**：增强立体感
- **圆角设计**：更加友好和现代
- **选中效果**：绿色系渐变 + 发光动画
- **悬停效果**：黄色系渐变 + 橙色边框

#### 连接线样式
- **平滑贝塞尔曲线**：三次贝塞尔曲线，更加流畅
- **柔和颜色**：灰色系，不抢眼
- **选中发光**：选中时绿色发光效果

#### 画布背景
- **渐变背景**：从浅蓝到浅灰的垂直渐变
- **网格参考**：30px 间距的浅色网格线
- **视觉引导**：便于对齐和定位

#### 动画效果
- **创建动画**：从父节点位置淡入并移动到目标位置
- **删除动画**：淡出并缩小到中心点
- **移动动画**：平滑移动到新位置
- **高亮动画**：选中时发光收缩效果
- **悬停动画**：鼠标悬停时即时反馈

### AI 辅助

- **智能对话**：内置 AI 聊天窗口，提供智能辅助
- **内容生成**：根据当前节点生成相关内容
- **结构建议**：AI 可以建议思维导图结构优化

## 技术栈

- **Vue 3.5.25** - 渐进式 JavaScript 框架
- **Vite 7.3.1** - 下一代前端构建工具
- **HTML5 Canvas** - 高性能 2D 绘图
- **JavaScript (ES6+)** - 现代化 JavaScript 特性

## 项目特点

### 性能优化
- **视锥剔除**：只渲染可见区域的节点
- **DPI 适配**：完美支持高 DPI 显示器
- **动画优化**：使用 requestAnimationFrame 实现流畅动画
- **文本截断**：自动处理过长文本

### 用户体验
- **流畅动画**：所有交互都有平滑的过渡效果
- **即时反馈**：鼠标悬停、选中都有视觉反馈
- **智能布局**：自动避免节点重叠
- **撤销重做**：支持 50 步历史记录

### 视觉设计
- **现代化界面**：采用现代设计语言
- **柔和配色**：护眼的配色方案
- **层次分明**：通过阴影和渐变增强层次感
- **细节精致**：圆角、阴影、发光等细节处理

### API 接口

```javascript
// 获取思维导图数据
const data = mindMapRef.value.getMindMapData()

// 加载思维导图数据
mindMapRef.value.loadMindMapData({ nodes, rootIds })

// 导出为 PNG 图片
const pngDataUrl = mindMapRef.value.exportAsPNG()

// 清空画布
mindMapRef.value.clearCanvas()

// 获取/设置布局方向
mindMapRef.value.getLayoutDirection()
mindMapRef.value.setLayoutDirection('left-right')
```
